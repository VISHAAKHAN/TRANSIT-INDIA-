Key Insight: SMS is the Dominant Cost Driver

Across all three tiers, AWS SNS SMS costs account for 55–85% of total infrastructure spend. At national scale, SMS alone costs $20,000/month (85% of total). This presents the single largest optimization opportunity—migrating to a hybrid SMS/Push Notification strategy could reduce overall costs by 50–60%. Without SMS, the pure infrastructure cost at national scale would be approximately $3,523/month—remarkably cost-effective for serving 500,000 daily users across all of India.

▼ Cost Optimization Strategies

Recommended strategies to reduce AWS costs while maintaining production-grade reliability and performance.

✉

Hybrid SMS + Push Notifications

Replace 70% of SMS notifications with Firebase Cloud Messaging (FCM) push notifications. Reserve SMS for OTP authentication and critical alerts only. FCM is free for unlimited messages.

Save up to $14,000/mo at Tier 3
⏱

Reserved Instances (1-year)

Purchase 1-year Reserved Instances for RDS and ElastiCache. Provides up to 40% savings on database costs compared to on-demand pricing with no upfront payment.

Save ~$1,500/mo at Tier 3
⚙

Lambda Power Tuning

Use AWS Lambda Power Tuning tool to find optimal memory configuration. Reducing from 512 MB to 256 MB for lightweight functions can cut Lambda costs by 40%.

Save ~$175/mo at Tier 3
☂

S3 Intelligent Tiering

Enable S3 Intelligent-Tiering for audit logs and ML training data buckets. Automatically moves infrequently accessed data to lower-cost storage classes.

Save ~$8/mo at Tier 3
⚡

CloudFront Caching Optimization

Increase cache TTL for static assets (route maps, stop data) and implement aggressive browser caching. Target 95%+ cache hit ratio to reduce origin requests.

Save ~$200/mo at Tier 3
★

SageMaker Serverless Inference

Switch from always-on SageMaker endpoints to Serverless Inference for ML models. Pay only when inference requests come in, ideal for variable traffic patterns.

Save ~$150/mo at Tier 3
▣

NAT Gateway Optimization

Use VPC endpoints for S3 and DynamoDB access to bypass NAT Gateway. Deploy Lambda in public subnets where possible. Consider NAT instances for dev environments.

Save ~$80/mo at Tier 3
⍑

API Gateway HTTP API Migration

Migrate from REST API to HTTP API in API Gateway. HTTP APIs are up to 71% cheaper ($1.00 vs $3.50 per million) and support JWT authorization natively.

Save ~$125/mo at Tier 3

i Assumptions & Notes

Key assumptions used in this cost estimation model.

  • All prices are based on AWS ap-south-1 (Mumbai) region as of 2025 and are in USD.
  • Lambda compute time estimated at average 500ms per invocation at 512 MB memory.
  • RDS Multi-AZ doubles the instance cost; each read replica costs the same as the primary.
  • ElastiCache cluster mode with 3 shards means 3x the single-node cost for Tier 3.
  • S3 costs include storage + request charges; data transfer to CloudFront is free.
  • CloudFront pricing uses India-specific edge location rates ($0.14/GB for first 10 TB).
  • SNS SMS costs assume transactional SMS at $0.02 per message for India.
  • SageMaker costs assume 24/7 always-on endpoints for real-time inference.
  • NAT Gateway charged at $45/month base + $0.06 per GB of data processed.
  • WAF costs include $5/ACL + $1/rule + $0.60/million requests inspected.
  • AWS Free Tier benefits applied only to Tier 1 (first 12 months).
  • Data transfer between AZs charged at $0.02/GB (Tier 3 only).
  • Costs exclude domain registration, SSL certificates (ACM is free), and developer tooling.
  • No AWS Savings Plans or Enterprise Discount Program (EDP) pricing applied.
  • INR conversion uses approximate rate of 1 USD = 83 INR.
  • Monthly costs assume consistent usage; actual costs may vary with traffic patterns.
