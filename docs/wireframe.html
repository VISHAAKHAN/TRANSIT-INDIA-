CAPTCHA validation is client-side against a pool of 8 predefined strings (case-insensitive, whitespace-ignored). On successful validation, api.sendOTP(mobile) triggers AWS SNS to deliver a 6-digit OTP. If SMS delivery fails, a debug OTP is returned for testing.
