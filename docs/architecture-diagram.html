API Endpoints

  • POST/api/v1/auth/send-otpSend OTP to phone
  • POST/api/v1/auth/verify-otpVerify OTP, return role
  • GET/api/v1/routes/searchSearch routes by from/to stops
  • GET/api/v1/routes/active-busesList all running buses
  • GET/api/v1/stopsList all stops
  • GET/api/v1/operator/routesList operator routes
  • GET/api/v1/operator/routes/{id}Route detail with stops
  • POST/api/v1/operator/trips/activateActivate a bus trip
  • GET/api/v1/operator/trips/activeList active trips
  • POST/api/v1/operator/trips/{id}/deactivateDeactivate a trip
  • POST/api/v1/safety/escalateReport safety incident
  • POST/api/v1/feedbackSubmit feedback
  • POST/api/v1/admin/seedSeed database
